우선 2020년도 서류만으로 존재하는 것은 진정한 포트폴리오가 아니라고 생각하여 혼자 만들어보겠다며 시작한 프로젝트 입니다. 사용한 언어는 HTML,CSS,JavaScript FrontEnd로 사용하고 있습니다. 또한, 사용한 라이브러리는 FrontEnd는 Bootstrap을 주로 사용했습니다. 현재 보시는 이 페이지에 해당합니다. Github 바로가기

프로젝트들은 최근 개발한 순으로 정렬되어 있습니다

Stacks

만들고자 하는 목표에 대해 필요한 기술들을 빠르게 습득하고 구현합니다. 할 수 있는 경계선을 최대한 더 넓혀 나가고자 합니다.
기술 구분 로고 활용 수준 평가
JavaScript JavaScript, ECMAScript, Webpack 과 Babel등을 위주로 개발하며 모듈화를 통한 플러그인 개발 및 SDK도 가능합니다.
브라우저 환경의 JavaScript의 경우 브라우저 호환성을 중요시하여 IE 환경에서도 정상 작동하도록 개발하고 있으며 최신 문법인 ES6 또한 자유자재로 사용하고 있습니다
Node.js Node.js로 Express, Koa등 웹 개발 프레임워크를 구축할 수 있으며, React와 MERN Stack 방식으로 조합하여 개발할 수 있습니다
Handlebars, Liquid와 같은 Template Engine을 View 형태로 제공하여 구축하거나 서버 로깅 등을 할 수 있습니다
Vue Redux와 Slice로 상태 관리를 하며 Axios를 주로 사용하여 API 요청을 구현합니다.
대부분의 필요한 웹 컴포넌트를 능숙하게 개발할 수 있습니다(CRUD, Formik, Yup 등..) 또한 Node.js로부터 JWT를 발급받아 사용자 인증을 구현할 수 있습니다
객관화하기 어려운 관계로, 제가 이해하고 구현할 수 있는 수준을 작성했습니다.

Front End

HTML,CSS,JavaScript를 가장 많이 사용해 왔으며 가장 자신 있는 언어입니다. 특히 Vue에 관심이 많습니다.

Back end

Node.js를 가장 많이 사용하며 SpringBoot 등의 언어를 활용하고 있습니다 Express.js, Koa를 공부해보고 싶습니다.

Data base

MySQL를 가장 좋아하며 MySQL를 주로 활용하고 있습니다. 물론 Firebase, MongoDB도 사용하는 중이며 MongoDB에 관심이 많습니다.

Utilities

GitHub, Notion, Slack 등 사용하고 있습니다. 더 많이 알수록 효율은 더 높아진다고 믿습니다.

Document

가장 중요한 자산은 태도라고 생각합니다. 문제를 대하는 태도, 개발을 대하는 태도, 팀원을 대하는 태도가 오랜 시간과 결합하면 엄청난 능력이 된다고 믿습니다. 그동안 노력해온 다양한 문서를 아래에서 보실 수 있습니다.
